二分查找 ​

给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

TIP

解答：首先介绍一下二分法模板，了解了二分法的模板之后，可以帮助我们处理各种边界问题。

二分法的两个模板分别用于找两个边界，如下图所示，要想找出左边红色的边界，需要令mid=l+r+12，要想找出右边绿色的边界，要让mid=l+r2（红色、绿色都表示一种条件）。其余的while判断都是相同的。

  • 对于目标第一次出现的位置，我们可以把区间划分为【<target, >=target】，这样右边区间的左边界即为所求
  • 对于目标最后一次出现的位置，我们可以把区间划分为【<=target, >target】，这样左边区间的右边界即为所求

二分模板如下：

c++
bool check(int x) {/* ... */} // 检查x是否满足某种性质

// 区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int bsearch_1(int l, int r)
{
    while (l < r)
    {
        int mid = l + r >> 1;
        if (check(mid)) r = mid;    // check()判断mid是否满足性质
        else l = mid + 1;
    }
    return l;
}
// 区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int bsearch_2(int l, int r)
{
    while (l < r)
    {
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}

class Solution {
    public int search(int[] nums, int target) {
        int i = 0, j = nums.length - 1;
        while (i < j) {
            int mid = i + j >> 1;
            if (nums[mid] >= target) {
                j = mid;
            } else {
                i = mid + 1;
            }
        }
        // 可能不存在，不存在返回-1
        return nums[i] == target ? i : -1;
    }
}

移除元素** ​

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

TIP

解法：双指针。这道题经常忘记，思想是快指针走在前面，如果快指针的元素不是val，那就赋给slow，然后slow才会往前1步。即：slow永远走在后面，当fast的值合适的时候，才会赋给slow，此时slow++

java
class Solution {
    public int removeElement(int[] nums, int val) {
        int fast = 0, slow = 0;
        for (; fast < nums.length; fast++) {
            if (nums[fast] != val) {
                nums[slow++] = nums[fast];
            }
        }
        return slow;
    }
}

有序数组的平方 ​

给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

TIP

解法：想象成一个二次函数，两端的是绝对值最大的。我们只需要从两边向中间遍历。

java
class Solution {
    public int[] sortedSquares(int[] nums) {
        int[] res = new int[nums.length];
        int index = res.length - 1;
        int l = 0, r = nums.length - 1;
        while (l <= r) {
            if (Math.abs(nums[l]) > Math.abs(nums[r])) {
                res[index--] = nums[l] * nums[l];
                l++;
            } else {
                res[index--] = nums[r] * nums[r];
                r--;
            }
        }
        return res;
    }
}

长度最小的子数组** ​

给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其总和大于等于 target 的长度最小的 连续子数组[numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

TIP

解法：双指针。这个双指针的思想很好理解，但是我觉得代码需要多注意，多体会。使用一个for循环，每次加上一个nums[r]，再使用一个while循环，每次减少一个。

java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int sum = 0;
        int res = Integer.MAX_VALUE;
        int l = 0;
        for (int r = 0; r < nums.length; r++) {
            // 每次先加一个
            sum += nums[r];
            // 如果当前和大于target，每次减一个
            while (sum >= target) {
                res = Math.min(r - l + 1, res);
                sum -= nums[l++];
            }
        }
        return res == Integer.MAX_VALUE ? 0 : res;
    }

}

螺旋矩阵 II ​

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

示例 1：

输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]
1
2

TIP

分别设置4个边界：left、right、up、down。然后在每次for遍历完一个边，把相应的边做++或--的操作，如果两个边越过了，就说明遍历结束了。

java
class Solution {
    public int[][] generateMatrix(int n) {
        int[][] res = new int[n][n];
        int l = 0, r = n - 1, u = 0, d = n - 1;
        int count = 1;
        while (true) {
            for (int j = l; j <= r; j++) {
                res[u][j] = count++;
            }
            if (++u > d) {
                break;
            }
            for (int i = u; i <= d; i++) {
                res[i][r] = count++;
            }
            if (--r < l) {
                break;
            }
            for (int j = r; j >= l; j--) {
                res[d][j] = count++;
            }
            if (--d < u) {
                break;
            }
            for (int i = d; i >= u; i--) {
                res[i][l] = count++;
            }
            if (++l > r) {
                break;
            }
        }
        return res;
    }
}

移除链表元素 ​

给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。

示例 1：

输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
1
2

TIP

解法：一般和链表删除元素有关的都会有一个dummy节点，建议与链表有关的题，都可以使用一个dummy，很方便

java
class Solution {
    public ListNode removeElements(ListNode head, int val) {
		
        ListNode dummy = new ListNode(-1, head);
        ListNode cur = dummy;
        while (cur.next != null) {
            if (cur.next.val == val) {
                cur.next = cur.next.next;
            } else {
                cur = cur.next;
            }
        }
        return dummy.next;
    }
}

设计链表 ​

实现 MyLinkedList 类：

  • MyLinkedList() 初始化 MyLinkedList 对象。
  • int get(int index) 获取链表中下标为 index 的节点的值。如果下标无效，则返回 -1 。
  • void addAtHead(int val) 将一个值为 val 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
  • void addAtTail(int val) 将一个值为 val 的节点追加到链表中作为链表的最后一个元素。
  • void addAtIndex(int index, int val) 将一个值为 val 的节点插入到链表中下标为 index 的节点之前。如果 index 等于链表的长度，那么该节点会被追加到链表的末尾。如果 index 比长度更大，该节点将 不会插入 到链表中。
  • void deleteAtIndex(int index) 如果下标有效，则删除链表中下标为 index 的节点。

TIP

解答：就是要自定义Node，然后实现几个api，不过需要细心。写一个dummy

java
public class MyLinkedList {

    int size = 0;
    ListNode dummy = new ListNode(-1, null);
    public MyLinkedList() {

    }

    public int get(int index) {
        if (index < 0 || index >= size) {
            return -1;
        }
        ListNode cur = dummy;
        while (index-- != 0) {
            cur = cur.next;
        }
        return cur.next.val;
    }

    public void addAtHead(int val) {
        this.addAtIndex(0, val);
    }

    public void addAtTail(int val) {
        this.addAtIndex(size, val);
    }

    public void addAtIndex(int index, int val) {
        if (index < 0 || index > size) {
            return;
        }
        ListNode cur = dummy;
        ListNode pre = dummy;
        while (index-- != 0) {
            pre = cur;
            cur = cur.next;
        }
        // 如果cur为空，说明是要在最后加上
        if (cur != null) {
            ListNode newNode = new ListNode(val, null);
            newNode.next = cur.next;
            cur.next = newNode;
        } else {
            // 在中间加上
            ListNode newNode = new ListNode(val, null);
            pre.next = newNode;
        }
        size++;
    }

    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            return;
        }
        ListNode cur = dummy;
        while (index-- != 0) {
            cur = cur.next;
        }
        cur.next = cur.next.next;
        size--;
    }
}

反转链表 ​

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

TIP

记录一个pre值，将cur用于遍历，每次cur的指针指回pre。接着再调整pre为cur，cur指向后面的元素。

java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        while (cur != null) {
            ListNode tmp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = tmp;
        }
        return pre;
    }
}

两两交换链表中的节点** ​

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

TIP

解法：使用一个dummy。这题经常忘记写法

java
class Solution {
    public ListNode swapPairs(ListNode head) {
        ListNode dummy = new ListNode(-1, head);
        ListNode cur = dummy;
        while (cur.next != null && cur.next.next != null) {
            ListNode cur1 = cur.next;
            ListNode cur2 = cur.next.next;
            cur.next = cur2;
            cur1.next = cur2.next;
            cur2.next = cur1;

            cur = cur1;
        }
        return dummy.next;
    }
}

删除链表的倒数第 N 个结点 ​

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例 1：

输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
1
2

TIP

解法：可以先遍历一次，得到链表的大小，再移动到合适的位置。也可以使用一个快慢指针，快指针先跑出一个差距，然后满指针再开始和快指针以一样的速度前进，保持距离。

java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy = new ListNode(-1, head);
        ListNode fast = dummy;
        ListNode slow = dummy;
        while (n-- != 0) {
            fast = fast.next;
        }
        while (fast.next != null) {
            slow = slow.next;
            fast = fast.next;
        }
        slow.next = slow.next.next;
        return dummy.next;
    }
}

链表相交 ​

给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 。

图示两个链表在节点 c1 开始相交

TIP

解法：以上题为例。A比B短1个长度，因此让B先往前一个长度，二者再同时出发，如果二者在途中相遇了，即为链表的交点。如果二者都为空值了，还没有相遇，就说明没有交点。

java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        int lenA = size(headA);
        int lenB = size(headB);
        // 这一步确保B是更长的
        if (lenA > lenB) {
            return getIntersectionNode(headB, headA);
        }
        int diff = lenB - lenA;
        while (diff-- != 0) {
            headB = headB.next;
        }
        while (headA != null && headB != null) {
            if (headA == headB) {
                return headA;
            }
            headA = headA.next;
            headB = headB.next;
        }
        return null;
    }
    private int size(ListNode node) {
        int len = 0;
        while (node != null) {
            len++;
            node = node.next;
        }
        return len;
    }
}

环形链表 II ​

给定一个链表的头节点 head ，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。

不允许修改 链表。

TIP

解法：设置快慢指针，快指针每次两个单位，慢指针每次走一个单位。二者一定会在圆环内相遇，而且是慢指针在跑第一圈的时候相遇。接着在meet的地方，head处同时设置一个指针开始前进。它们相遇的地方就是入口

java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode fast = head;
        ListNode slow = head;
        ListNode meet = null;
        while (fast != null && fast.next != null) {
            fast = fast.next.next;
            slow = slow.next;
            if (fast == slow) {
                meet = slow;
                break;
            }
        }
        // 如果没有相遇，说明没有环
        if (meet == null) {
            return null;
        }
        // 在相遇的地方、head处同时出发一个指针
        ListNode cur = head;
        while (head != meet) {
            head = head.next;
            meet = meet.next;
        }
        return head;
    }
}

有效的字母异位词 ​

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

注意：若 s 和 t 中每个字符出现的次数都相同，则称 s 和 t 互为字母异位词。

TIP

1、遍历字符串s，统计各个字符出现的个数

2、遍历字符串t，做自减操作；如果出现负数，可以提前返回false。

3、遍历hash，判断是否有不为0的元素

java
class Solution {
    public boolean isAnagram(String s, String t) {
        int[] hash = new int[26];
        for (char c : s.toCharArray()) {
            hash[c - 'a']++;
        }
        for (char c : t.toCharArray()) {
            if (--hash[c - 'a'] < 0) {
                return false;
            }
        }
        for (int c : hash) {
            if (c != 0) {
                return false;
            }
        }
        return true;
    }
}

两个数组的交集 ​

给定两个数组 nums1 和 nums2 ，返回 它们的 交集 。输出结果中的每个元素一定是 唯一 的。我们可以 不考虑输出结果的顺序 。

TIP

1、遍历nums1，记录nums1中出现的数字

2、遍历num2，判断数字在hash中出现了没。如果出现，加入set。

3、将set转为int[]返回

java
class Solution {
    public int[] intersection(int[] nums1, int[] nums2) {
        Set<Integer> set = new HashSet<>();
        boolean[] hash = new boolean[1001];
        for (int i = 0; i < nums1.length; i++) {
            hash[nums1[i]] = true;
        }
        for (int i = 0; i < nums2.length; i++) {
            if (hash[nums2[i]]) {
                set.add(nums2[i]);
            }
        }
        int index = 0;
        int[] res = new int[set.size()];
        Iterator<Integer> it = set.iterator();
        while (it.hasNext()) {
            res[index++] = it.next();
        }
        return res;
    }
}

快乐数 ​

编写一个算法来判断一个数 n 是不是快乐数。

「快乐数」 定义为：

  • 对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
  • 然后重复这个过程直到这个数变为 1，也可能是 无限循环 但始终变不到 1。
  • 如果这个过程 结果为 1，那么这个数就是快乐数。

如果 n 是 快乐数 就返回 true ；不是，则返回 false 。

TIP

用一个set集合来存储循环中存储的元素，在循环过程中，

java
class Solution {
    public boolean isHappy(int n) {
        Set<Integer> set = new HashSet<>();
        while (true) {
            if (!set.add(n)) {
                return false;
            }
            n = getSquare(n);
            if (n == 1) {
                return true;
            }
        }
    }
    private int getSquare(int n) {
        int res = 0;
        while (n > 0) {
            int last = n % 10;
            res += last * last;
            n /= 10;
        }
        return res;
    }
}

两数之和 ​

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

TIP

方法一：可以使用两个for循环嵌套，来判断是否有相应的整数

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            if (map.containsKey(target - nums[i])) {
                return new int[]{map.get(target - nums[i]), i};
            }
            map.put(nums[i], i);
        }
        return new int[]{};
    }
}

四数相加 II ​

给你四个整数数组 nums1、nums2、nums3 和 nums4 ，数组长度都是 n ，请你计算有多少个元组 (i, j, k, l) 能满足：

  • 0 <= i, j, k, l < n
  • nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0

TIP

方法一：使用4个for循环，时间复杂度O(n^4)，不推荐会超时

方法二：哈希表，先用两个for循环记录外层的哈希表，再用两个外层的for循环"回头"寻找

java
class Solution {
    public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int num1 : nums1) {
            for (int num2 : nums2) {
                map.put(num1 + num2, map.getOrDefault(num1 + num2, 0) + 1);
            }
        }
        int res = 0;
        for (int num3 : nums3) {
            for (int num4 : nums4) {
                res += map.getOrDefault(-num3 - num4, 0);
            }
        }
        return res;
    }
}

赎金信 ​

给你两个字符串：ransomNote 和 magazine ，判断 ransomNote 能不能由 magazine 里面的字符构成。

如果可以，返回 true ；否则返回 false 。

magazine 中的每个字符只能在 ransomNote 中使用一次。

java
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        int[] hash = new int[26];
        for (char c : ransomNote.toCharArray()) {
            hash[c - 'a']++;
        }
        for (char c : magazine.toCharArray()) {
            hash[c - 'a']--;
        }
        for (int c : hash) {
            if (c > 0) {
                return false;
            }
        }
        return true;
    }
}

三数之和 ​

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请

你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

java
class Solution {

    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 2; i ++) {
            // 剪枝操作，如果当前nums[i]已经大于0，说明后续不可能存在符合条件的集合
            if (nums[i] > 0) {
                break;
            }
            // 如果和上一个元素相等，就下一步，否则会出现重复
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            int l = i + 1, r = nums.length - 1;
            while (l < r) {
                int sum = nums[i] + nums[l] + nums[r];
                if (sum < 0) {
                    l++;
                } else if (sum > 0) {
                    r--;
                } else {
                    List<Integer> list = new ArrayList<>();
                    list.add(nums[i]);
                    list.add(nums[l]);
                    list.add(nums[r]);
                    res.add(list);
                    while (l < r && nums[l] == nums[l + 1]) {
                        l++;
                    }
                    while (l < r && nums[r] == nums[r - 1]) {
                        r--;
                    }
                    l++;
                    r--;
                }
            }
        }
        return res;
    }
}

四数之和 ​

给你一个由 n 个整数组成的数组 nums ，和一个目标值 target 。请你找出并返回满足下述全部条件且不重复的四元组 [nums[a], nums[b], nums[c], nums[d]] （若两个四元组元素一一对应，则认为两个四元组重复）：

  • 0 <= a, b, c, d < n
  • a、b、c 和 d 互不相同
  • nums[a] + nums[b] + nums[c] + nums[d] == target

TIP

在三数之和中，是固定了第一个元素，动态的移动第二个、第三个元素。

在四数之和中，我们固定前两个元素，动态的移动第三个、第四个元素，同样需要一些剪枝的操作。

java
class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 3; i++) {
            if (nums[i] > target && nums[i] >= 0) {
                break;
            }
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            for (int j = i + 1; j < nums.length - 2; j++) {
                if (nums[i] + nums[j] > target && nums[j] >= 0) {
                    break;
                }
                if (j > i + 1 && nums[j] == nums[j - 1]) {
                    continue;
                }
                int left = j + 1, right = nums.length - 1;
                while (left < right) {
                    int sum = nums[i] + nums[j] + nums[left] + nums[right];
                    if (sum < target) {
                        left++;
                    } else if (sum > target) {
                        right--;
                    } else {
                        List<Integer> list = new ArrayList<>();
                        list.add(nums[i]);
                        list.add(nums[j]);
                        list.add(nums[left]);
                        list.add(nums[right]);
                        res.add(list);
                        while (left < right && nums[left] == nums[left + 1]) {
                            left++;
                        }
                        while (left < right && nums[right] == nums[right - 1]) {
                            right--;
                        }
                        left++;
                        right--;
                    }
                }
            }
        }
        return res;
    }
}

反转字符串 ​

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 s 的形式给出。

不要给另外的数组分配额外的空间，你必须**原地修改输入数组**、使用 O(1) 的额外空间解决这一问题。

java
class Solution {
    public void reverseString(char[] s) {
        int i = 0, j = s.length - 1;
        while (i < j) {
            char tmp = s[i];
            s[i] = s[j];
            s[j] = tmp;
            i++;
            j--;
        }
        
    }
}
