JETBRA系列IDE安装 ​

网站：https://3.jetbra.in/

一、下载script ​

进入网站，挑选一个可用网站，可能需要多试几次

二、install操作 ​

在scripts文件夹下，执行uninstall-current-user.vbs（防止以前用过install操作），再执行install-current-user.vbs，等待done结束即可。

三、进入ide ​

在提示需要激活的页面，将网站中的active code复制进去即可。

四、idea的一些使用技巧 ​

maven设置 ​

idea是自带maven的，如果你不设置本地的maven，则会自动使用idea自带的maven，修改maven配置的地方：

注释 ​

idea注释的时候，是会从行首开始的，这样当我们代码有缩进的时候，很不方便。以Java为例：点击Editor->Code Style -> Java，找到 Code Generation，把右下角的设置修改

忽略大小写提示 ​

在Java中，类一般是大驼峰命名，提示的时候我们也需要使用大小写切换，很不方便。找到Editor->General->Code Completion，修改Match-case
